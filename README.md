# AI Learning Agent System

A lightweight AI-powered learning workflow system designed to enhance knowledge acquisition, structuring, and retrieval efficiency.

## 🚀 Overview

Traditional learning workflows suffer from fragmented knowledge, low reusability, and lack of feedback loops. This project introduces an AI-driven agent-based system to transform raw learning materials into structured, queryable knowledge.

The system integrates LLMs with a structured note-taking workflow (e.g., Obsidian) to build a personal knowledge graph.

---

## 🧠 Core Features

- 📚 **Content Structuring**
  - Extract key ideas, concepts, and reasoning chains from raw text
  - Transform unstructured content into structured knowledge units

- 🔗 **Atomic Knowledge System**
  - Automatically split content into atomic notes
  - Build connections between concepts (graph-based learning)

- 🤖 **Agent-based Workflow**
  - Multi-step reasoning pipeline:
    - Summarization Agent
    - Concept Extraction Agent
    - Critique / Reflection Agent

- 🧩 **Human-in-the-loop Design**
  - AI handles structure & compression
  - User handles judgment & refinement

- 🧪 **Active Recall Support**
  - Auto-generated questions
  - Self-testing workflow integration
